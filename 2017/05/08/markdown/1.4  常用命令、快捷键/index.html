cd命令

cd命令（change directory）

  • cd 回车 进入当前的家目录
  • cd + 路径 进入该目录
  • cd - 回到上一次所在的目录 （交替）
  • cd · == cd · 代表 /root 相当于家目录
  • cd. 当前目录
  • cd.. 进入前一级目录

查看当前用户

  • whoami 查看当前用户
  • id 查看当前用户
  • echo $HOME 查看当前目录 $HOME 是系统自带的变量 表示家目录
  • pwd 查看当前在哪个目录下 echo $PWD 查看当前目录

常用 cd + 路径

> cd /usr/local/src

ls命令

ls命令（list）

  • ls 回车 直接列出当前目录下的所有文件和目录
  • ls -l 显示文件的详细信息 属性
  • ls -a 查看隐藏文件（查看全部文件包括隐藏文件）
  • ls -d 只查看当前目录 针对目录
  • ls -ld 查看目录本身的详细信息
  • ls -lt 按时间排序
  • ls -i 查看inode 号 （唯一的 每个文件都有一个）

常用

ls -lhtr （推荐使用，直接记住）

> ls -lhtr 1.txt
