创建和删除目录

创建目录命令

mkdir

/tmp创建下创建一个123目录

mkdir /tmp/123

注：绝对路径和相对路径

可以使用 man 命令来查看命令的帮助文档

man mkdir

  • -p 自动创建不存在的子目录
  • -v 表示创建过程可视化

常用形式：

mkdir -pv

删除目录命令

rmdir

常用形式：

rmdir -pv

注：表示删除全部，但不能删除非空目录，不能删除文件

查看目录结构

tree

安装命令

> yum -y install tree

rm命令

可以删除目录，文件

常用形式：

rm -frv

  • -f 强制删除
  • -r 删除目录
  • -v 可视化

cp命令

用于拷贝文件

cp /root/1.txt /root/2.txt

  • -r 拷贝目录

mv命令

用于移动文件、目录或者重命名

移动：

　mv /root/1.txt /tmp/

重命名：

mv /root/1.txt 2.txt

文件查看和创建

创建文件

touchu 1.txt

vi 1.txt

查看文件

  • cat —— 读取文件内容
  • tac —— 重最后一行读取
  • more —— 重头显示，使用空格回车向下翻，不能向上翻
  • head -2 表示只看前2行
  • tail -2 表示只看最后2行
  • tail -f 动态查看最后内容

一般建议直接使用编辑命令vi、vim来查看文档。直接查看多用cat
